Breathing Room Calculator

The Breathing Room Calculator is a JavaScript program I wrote to calculate how much money you would have left over each month after paying your bills. It was my first attempt at combining JavaScript with HTML.

Maze Navigation Program

I created a program for an Arduino-based "Sparki" bot that allowed it to successfully navigate around random obstacles without hitting them, and edited together a video to demonstrate it when the project was turned in.
